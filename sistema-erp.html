Sistema Web (ERP)

Punto de venta · Inventarios · Usuarios · Seguridad

Sistema web empresarial desarrollado desde cero con HTML, CSS, JavaScript, jQuery, PHP y MySQL. Incluye control de inventarios, clientes, proveedores, ventas, cotizaciones y gestión de usuarios con 3 niveles de acceso. Implementa seguridad avanzada con contraseñas robustas, caducidad por tiempo, reCAPTCHA y conexión SSL.
